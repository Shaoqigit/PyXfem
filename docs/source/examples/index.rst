运行示例
--------

所有示例位于 ``tests/`` 目录:

.. code-block:: bash

   # 运行特定测试
   python tests/test_2D_air.py
   
   # 运行所有测试
   python run_test.py

示例输出
--------

每个示例通常会生成:

- 数值结果 (压力场、SPL等)
- 可视化图形
- 与解析解的对比 (如果有)

自定义示例
----------

基于现有示例创建自己的仿真:

1. 复制一个类似示例
2. 修改几何和材料参数
3. 调整边界条件
4. 运行并分析结果
